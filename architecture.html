Factory Inventory Management System

Vue 3 + FastAPI demo with in-memory mock data

System Architecture

Frontend

Vue 3 + Vite · port 3000
  • Composition API
  • vue-router
  • axios client
  • Custom SVG charts
  • i18n composable
→

Backend

FastAPI + uvicorn · port 8001
  • Pydantic validation
  • Query-param filters
  • CORS enabled
  • Auto OpenAPI docs
→

Data

JSON files · in-memory
  • inventory.json (32)
  • orders.json (~250)
  • demand_forecasts.json
  • spending.json
  • Loaded at startup

Data Flow

1. User filtersComposable useFilters holds reactive refs (warehouse, category, status, month)
2. API callclient/src/api.js builds query params, axios GET to localhost:8001/api/*
3. Filter + validateapply_filters() filters in-memory list, Pydantic models validate response shape
4. RenderRaw data in ref(), derived values in computed(), tables/charts update reactively
